# Prerequisites

Before setting up Stacker locally, ensure you have the following installed:

## Required Software

- **Node.js 22** — JavaScript runtime ([download](https://nodejs.org/))
- **Yarn 4** — Package manager (installed via `corepack enable`)
- **MariaDB** — Relational database ([download](https://mariadb.org/download/))
- **Docker** — Container runtime, required for integration tests ([download](https://www.docker.com/products/docker-desktop/))

## Recommended Tools

- **TablePlus** or **DBeaver** — Database GUI client
- **VS Code** or **WebStorm** — IDE with TypeScript support

## Verify Installation

```bash
node --version    # v22.x.x
yarn --version    # 4.x.x
docker --version  # Docker version 27.x.x
mariadb --version # mariadb Ver 11.x.x
```
