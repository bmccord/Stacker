# Stacker

Stacker is a production-ready GraphQL + React starter kit built with modern TypeScript. It provides a complete foundation for building full-stack web applications with authentication, authorization, and CRUD operations.

## What's Included

- **GraphQL API** — Apollo Server 4, Express, Prisma 7, MariaDB
- **React UI** — Vite, Apollo Client, shadcn/ui, Tailwind CSS, React Hook Form
- **CLI Tool** — Data export and seed generation
- **Documentation** — This Docusaurus site
- **CI/CD** — GitHub Actions with multi-tier deployment
- **Docker** — Production-ready containers with nginx

## Example App: Bookshelf

The starter kit includes a fully functional bookshelf/library application demonstrating:

- Books with authors, genres, descriptions, and cover images
- Star ratings and text reviews
- User management with invitation workflow
- Permission-based access control with customizable groups
- Dashboard with aggregate statistics
