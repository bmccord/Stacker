# UI Overview

The Stacker UI is a React single-page application built with Vite and TypeScript.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 19 |
| Build Tool | Vite |
| Language | TypeScript |
| Data | Apollo Client (GraphQL) |
| Components | shadcn/ui + Tailwind CSS |
| Forms | React Hook Form |
| Routing | React Router v7 |

## Key Features

- **Lazy-loaded pages** for optimal bundle splitting
- **Apollo Client** with automatic cache updates
- **shadcn/ui** component library built on Radix UI primitives
- **Tailwind CSS** for utility-first styling
- **React Hook Form** with Controller and useFieldArray for complex forms
- **Dark mode support** via CSS custom properties
